## Key Invariants
```
invariant 0 <= i <= |a|
invariant 0 <= j <= |b|
invariant |result| == i + j
invariant multiset(result) == multiset(a[..i]) + multiset(b[..j])
invariant SortedSeq(result)

```

## Key Assertions
- `a[..i] == a[..i-1] + [a[i-1]]` and `b[..j] == b[..j-1] + [b[j-1]]` for multiset steps.
- `a[..i] == a` and `b[..j] == b` at loop end.
