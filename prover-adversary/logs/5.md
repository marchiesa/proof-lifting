## Strategy

Build a histogram counting occurrences. The challenge is relating the map to the recursive `CountIn` function at each step.

### Attempt 1 (VERIFIED)
- Loop invariant tracks that `hist` is correct for the prefix `s[..i]`.
- Two helper lemmas:
  - `CountInAppend`: proves `CountIn(s + [elem], x) == CountIn(s, x) + (if elem == x then 1 else 0)` by structural induction.
  - `CountInNotIn`: proves `CountIn(s, x) == 0` when `x !in s`.
- In the loop body, when key is already in hist: use `CountInAppend` to show counts for all existing keys are unchanged (via `forall` statement) except the current key.
- When key is new: use `CountInNotIn` to show the count was 0, and `CountInAppend` for the increment.

## Key Invariants
```
invariant 0 <= i <= |s|
invariant forall x :: x in hist <==> x in s[..i]
invariant forall x :: x in hist ==> hist[x] == CountIn(s[..i], x)
```

## Helper Lemmas
- `CountInAppend(s, elem, x)`: relates CountIn of `s + [elem]` to CountIn of `s`
- `CountInNotIn(s, x)`: proves count is 0 when element is absent
