## Axioms Used

1. **Kahn's full output implies topological order** (line ~223): When Kahn's algorithm processes all n nodes (|result| == n), the result is a valid topological ordering. This requires showing that the BFS order respects all edges — each node is dequeued only after all its predecessors, which follows from the in-degree tracking.

2. **Kahn's stalling implies cycle exists** (line ~229): When Kahn's algorithm cannot process all nodes (|result| < n), a cycle exists in the graph. This is because the remaining nodes all have positive in-degree from within the remaining subgraph, which by a pigeonhole/walk argument implies a cycle.

## Design Decisions

- **`fuel` variable for termination**: The BFS loop's natural termination metric (|result| increasing or queue shrinking) is hard to express since queue can grow when neighbors are enqueued. A fuel counter bounded by n provides simple termination.

- **`fuel` must be non-ghost**: Since the while condition includes `fuel > 0`, making fuel ghost would put the entire loop body in ghost context, preventing assignments to non-ghost variables (queue, result, inDeg). Changed from `ghost var fuel` to `var fuel`.

- **`inDeg` as `int[]` not `nat[]`**: Decrementing in-degrees can temporarily go below zero in theory (though Kahn's algorithm prevents this), and nat subtraction would wrap to zero instead of going negative. Using `int[]` avoids this issue.

- **Added `requires` clause to `IsTopologicalOrder`**: The original spec's `IsTopologicalOrder` calls `PositionOf(order, v)` where `v in adj[u]`, but `PositionOf` requires `v in order`. This is only provable if `v < n` (from WellFormedGraph) and `IsPermutation(order, n)`. Added `requires forall u :: 0 <= u < n ==> forall k :: 0 <= k < |adj[u]| ==> adj[u][k] < n` to make the predicate well-formed.

## Errors Encountered

- **Ghost context error**: `ghost var fuel` made `fuel > 0` a ghost expression, putting the entire BFS loop body in ghost context. Fixed by making fuel non-ghost.
- **PositionOf precondition**: `v in adj[u]` didn't imply `v in order` without knowing `v < n`. Fixed by adding requires clause to `IsTopologicalOrder`.
- **Termination**: Natural decreases expressions for BFS were hard to formulate. Fuel approach solved this simply.
