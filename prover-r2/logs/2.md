## Key Invariants/Lemmas Added

1. **`SubseqDecompose` lemma**: Any IS of length > 1 can be decomposed into a prefix (IS ending at some earlier index) and the last element.

2. **`SubseqExtend` lemma**: Appending an index to a valid IS (with correct ordering) produces a valid IS.

3. **`SingletonIS` lemma**: `[i]` is always a valid IS ending at `i`.

4. **`ISEndingAt0HasLen1` lemma**: Any IS ending at index 0 must have length exactly 1 (since indices are nats, nothing can precede 0).

5. **`UpperBoundLemma`**: If dp[j] upper-bounds all IS ending at j for j < i, and dp[i] >= dp[j]+1 for all valid predecessors, then dp[i] upper-bounds all IS ending at i.

6. **`ProcessCell` helper method**: Factored out the processing of each DP cell (i > 0) into a separate method to reduce quantifier complexity. This was the key insight that resolved timeouts.

7. **`IsDPOptimalAt` maintained incrementally**: For each i, we prove both the existence (via ghost witness) and upper bound (via induction on IS decomposition).

8. **Ghost variable `dpOpt`**: Tracks the sequence of dp values for processed indices, passed to ProcessCell for proving correctness.
