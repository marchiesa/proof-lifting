# Problem 2: Power of 2 check

## Result: VERIFIED (attempt 1 of 10)

## Problem
Check if a bv8 is a power of 2 using `x != 0 && x & (x-1) == 0`.
Spec says there exists k in [0,8) such that x == 1 << k.

## Why Z3 struggles
The existential quantifier in IsPowerOf2Spec needs a witness. Connecting the algebraic bit trick to the existential requires enumerating all 8 bit positions.

## Solution approach

Two lemmas proving both directions of the equivalence:

1. `PowerOf2ImpliesBitTrick`: If x is a power of 2 (exists k where x == 1 << k), then x != 0 and x & (x-1) == 0. Uses a witness variable to bind k, then asserts the disjunction of all 8 cases. Z3 can verify each case concretely.

2. `BitTrickImpliesPowerOf2`: If x != 0 and x & (x-1) == 0, then x is a power of 2. Explicitly enumerates all 8 powers of 2 (1, 2, 4, 8, 16, 32, 64, 128) using if-else chain, providing the existential witness via `assert x == Shift1Left(k)`. The else branch asserts false, which Z3 verifies because no other bv8 value satisfies the precondition.

In the method body, we call the appropriate lemma depending on the computed result, and handle the `!result && x != 0` case with a proof by contradiction.
