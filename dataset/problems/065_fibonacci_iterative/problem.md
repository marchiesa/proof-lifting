# Fibonacci (Iterative)

## Description

Compute the n-th Fibonacci number iteratively. F(0) = 0, F(1) = 1, F(n) = F(n-1) + F(n-2).

## Input

- `n`: a non-negative integer

## Output

The n-th Fibonacci number.

## Examples

- `Fib(0)` returns `0`
- `Fib(1)` returns `1`
- `Fib(5)` returns `5`
- `Fib(10)` returns `55`
