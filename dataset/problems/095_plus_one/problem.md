# Plus One

## Description

Given a number represented as an array of digits, increment the number by one.
Each element in the array is a single digit (0-9). Handle carry propagation.

## Input

- `digits`: a non-empty sequence of valid digits (0-9)

## Output

A sequence `result` representing the number plus one, with carries propagated.

## Examples

- `PlusOne([1, 2, 3])` returns `[1, 2, 4]`
- `PlusOne([1, 2, 9])` returns `[1, 3, 0]`
- `PlusOne([9, 9, 9])` returns `[1, 0, 0, 0]`
- `PlusOne([0])` returns `[1]`
