# Maximum Depth of Nested Parentheses

## Description

Given a sequence of characters (represented as integers where 1 = '(' and 2 = ')'), find the maximum nesting depth of parentheses.

## Input

- `s`: a sequence of integers (1 for '(', 2 for ')')

## Output

The maximum nesting depth. Returns 0 for empty or balanced-at-zero input.
