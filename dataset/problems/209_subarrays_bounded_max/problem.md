# Number of Subarrays with Bounded Maximum

## Description

Given a sequence of positive integers and bounds L and R, count the number of contiguous subarrays where the maximum element is between L and R (inclusive).

## Input

- `a`: sequence of positive integers
- `L`, `R`: bounds with 1 <= L <= R

## Output

Count of subarrays whose maximum is in [L, R].
