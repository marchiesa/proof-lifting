# Find First and Last Position of Target in Sorted Array

## Description

Given a sorted array of integers and a target value, find the first and last position (index) where the target appears. Return (-1, -1) if the target is not found.

## Input

- `a`: a sorted sequence of integers
- `target`: the integer to search for

## Output

A pair (first, last) of indices where target first and last appears, or (-1, -1).
