# Zigzag Traversal of Matrix

## Description

Traverse a matrix row by row in zigzag order: even rows left-to-right,
odd rows right-to-left.

## Input

- `m`: a rectangular matrix
- `rows`, `cols`: dimensions

## Output

A flat sequence of all elements in zigzag order with length `rows * cols`.

## Examples

- `ZigzagTraversal([[1,2,3],[4,5,6]], 2, 3)` returns `[1,2,3,6,5,4]`
