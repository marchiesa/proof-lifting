# Longest Common Substring

## Description

Find the length of the longest common substring between two character sequences.

## Input

- `a`, `b`: two sequences of characters

## Output

Length of the longest common substring. 0 if no common substring exists.

## Examples

- `LongestCommonSubstring("abcde", "xbcy")` returns `2` (common: "bc")
- `LongestCommonSubstring("abc", "xyz")` returns `0`
