# Maximum Consecutive Ones

## Description

Given a sequence of integers, find the maximum number of consecutive 1s.

## Input

- `a`: a sequence of integers

## Output

An integer `maxCount` equal to the length of the longest contiguous run of 1s.

## Examples

- `MaxConsecutiveOnes([1, 1, 0, 1, 1, 1])` returns `3`
- `MaxConsecutiveOnes([1, 0, 1, 0, 1])` returns `1`
- `MaxConsecutiveOnes([])` returns `0`
