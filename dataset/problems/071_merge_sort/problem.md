# Merge Sort

## Description

Sort a sequence of integers using merge sort. The result must be sorted and be a permutation of the input (multiset equality).

## Input

- `s`: a sequence of integers

## Output

A sorted sequence that is a permutation of the input.

## Examples

- `MergeSort([3, 1, 4, 1, 5])` returns `[1, 1, 3, 4, 5]`
- `MergeSort([])` returns `[]`
- `MergeSort([1])` returns `[1]`
