# Minimum Domino Rotations

## Description

Given two sequences of equal length representing the top and bottom values of dominoes, find the minimum number of rotations so that all values in one row are the same. Return -1 if impossible.

## Input

- `tops`: sequence of integers (top face values)
- `bottoms`: sequence of integers (bottom face values)

## Output

Minimum rotations, or -1 if impossible.
