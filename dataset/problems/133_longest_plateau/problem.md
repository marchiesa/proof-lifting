# Longest Plateau

## Description

Find the length of the longest run of equal consecutive elements in a sequence.

## Input

- `a`: a non-empty sequence of integers

## Output

The length of the longest plateau (run of equal elements).

## Examples

- `LongestPlateau([1, 1, 2, 2, 2, 3])` returns `3`
- `LongestPlateau([1, 2, 3])` returns `1`
- `LongestPlateau([5, 5, 5, 5])` returns `4`
