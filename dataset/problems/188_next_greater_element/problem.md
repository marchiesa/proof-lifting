# Next Greater Element

## Description

For each element in a sequence, find the next element to the right that is strictly greater. If no such element exists, the answer is -1.

## Input

- `a`: a sequence of integers

## Output

A sequence `result` of the same length where result[i] is the first element a[j] with j > i and a[j] > a[i], or -1 if none exists.
