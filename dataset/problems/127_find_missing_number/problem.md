# Find Missing Number

## Description

Given a sequence of n distinct numbers from [0..n], find the missing number.

## Input

- `a`: sequence of n distinct integers from [0..n]
- `n`: the range bound

## Output

The integer in [0..n] not present in `a`.

## Examples

- `FindMissing([1, 2, 0], 3)` returns `3`
- `FindMissing([0, 2], 2)` returns `1`
- `FindMissing([], 0)` returns `0`
