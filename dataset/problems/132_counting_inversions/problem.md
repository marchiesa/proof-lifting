# Counting Inversions

## Description

Count the number of inversions in a sequence. An inversion is a pair (i, j)
where i < j but a[i] > a[j].

## Input

- `a`: a sequence of integers

## Output

The number of inversions.

## Examples

- `CountInversions([3, 1, 2])` returns `2` (inversions: (3,1), (3,2))
- `CountInversions([1, 2, 3])` returns `0`
- `CountInversions([3, 2, 1])` returns `3`
