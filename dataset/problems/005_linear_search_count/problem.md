# Count Occurrences

## Description

Given a sequence of integers and a target value, count the number of times the
target appears in the sequence.

## Input

- `a`: a sequence of integers
- `target`: the integer to count

## Output

An integer `count` equal to the number of indices `i` where `a[i] == target`.

## Examples

- `CountOccurrences([1, 3, 5, 3, 3, 7], 3)` returns `3`
- `CountOccurrences([1, 2, 3], 4)` returns `0`
- `CountOccurrences([], 1)` returns `0`
- `CountOccurrences([7, 7, 7], 7)` returns `3`
