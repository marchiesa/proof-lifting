# Array Intersection (Two Sorted Arrays)

## Description

Given two strictly sorted (no duplicates) integer sequences, compute their intersection:
the sequence of elements that appear in both.

## Input

- `a`: a strictly sorted sequence of integers
- `b`: a strictly sorted sequence of integers

## Output

A strictly sorted sequence `result` containing exactly the elements common to both `a` and `b`.

## Examples

- `Intersection([1, 3, 5, 7], [2, 3, 5, 8])` returns `[3, 5]`
- `Intersection([1, 3, 5], [2, 4, 6])` returns `[]`
- `Intersection([1, 2, 3], [1, 2, 3])` returns `[1, 2, 3]`
