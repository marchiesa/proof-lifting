# Count Range Sum

## Description

Count the number of contiguous subarrays whose sum falls within [lo, hi].

## Input

- `a`: sequence of integers
- `lo`, `hi`: range bounds with lo <= hi

## Output

Count of subarrays with sum in [lo, hi].

## Examples

- `CountRangeSum([-1, 2, 1], 0, 3)` returns `5`
