# Count Elements Greater Than Threshold

## Description

Count the number of elements in a sequence that are strictly greater than a given threshold.

## Input

- `a`: a sequence of integers
- `threshold`: an integer threshold

## Output

A natural number `count` equal to the number of elements in `a` that are strictly greater than `threshold`.

## Examples

- `CountGreaterThan([1, 5, 3, 7, 2], 3)` returns `2`
- `CountGreaterThan([10, 20, 30], 0)` returns `3`
- `CountGreaterThan([1, 2, 3], 10)` returns `0`
- `CountGreaterThan([], 5)` returns `0`
