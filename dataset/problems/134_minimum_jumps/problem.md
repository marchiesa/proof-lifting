# Minimum Number of Jumps

## Description

Given a sequence where a[i] represents the maximum jump length from position i,
determine the minimum number of jumps needed to reach the last position.

## Input

- `a`: a non-empty sequence of non-negative integers

## Output

The minimum number of jumps to reach the last position (0 for single element).

## Examples

- `MinJumps([2, 3, 1, 1, 4])` returns `2`
- `MinJumps([0])` returns `0`
