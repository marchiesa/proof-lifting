# Count Inversions (O(n^2))

## Description

Count the number of inversions in a sequence. An inversion is a pair (i, j) where i < j but a[i] > a[j].

## Input

- `a`: a sequence of integers

## Output

The number of inversions in the sequence.

## Examples

- `CountInversions([2, 1])` returns `1`
- `CountInversions([1, 2, 3])` returns `0`
- `CountInversions([3, 2, 1])` returns `3`
- `CountInversions([1, 3, 2, 4])` returns `1`
