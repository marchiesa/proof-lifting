# Check if Matrix is Symmetric

## Description

Check if a square matrix is symmetric (m[i][j] == m[j][i] for all i, j).

## Input

- `m`: a square matrix represented as `seq<seq<int>>`

## Output

`true` if the matrix is symmetric, `false` otherwise.

## Examples

- `CheckSymmetric([[1, 2], [2, 3]])` returns `true`
- `CheckSymmetric([[1, 2], [3, 4]])` returns `false`
