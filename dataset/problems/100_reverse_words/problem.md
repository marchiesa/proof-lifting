# Reverse Words in a Sentence

## Description

Given a sequence of "words" (each word is a sequence of integers),
reverse the order of the words.

## Input

- `words`: a sequence of sequences of integers

## Output

A sequence `result` where the words appear in reverse order.

## Examples

- `ReverseWords([[1,2],[3,4],[5,6]])` returns `[[5,6],[3,4],[1,2]]`
- `ReverseWords([[42]])` returns `[[42]]`
- `ReverseWords([])` returns `[]`
