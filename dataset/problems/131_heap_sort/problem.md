# Heap Sort

## Description

Sort an array in-place using a selection-based approach (heap sort variant):
repeatedly find the maximum in the unsorted portion and swap it to the end.

## Input

- `a`: an array of integers

## Output

The array sorted in non-decreasing order, as a permutation of the original.

## Examples

- `HeapSort([3, 1, 4, 1, 5])` produces `[1, 1, 3, 4, 5]`
