# Maximum Circular Subarray Sum

## Description

Given a non-empty array of integers, find the maximum subarray sum considering circular wrapping. The result is the maximum of: (1) the standard max subarray sum (Kadane's), and (2) the total sum minus the minimum subarray sum (the "wrap-around" case). If all elements are negative, return the maximum element.

## Input

- `a`: a non-empty sequence of integers

## Output

An integer representing the maximum circular subarray sum.
