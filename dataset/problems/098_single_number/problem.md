# Single Number

## Description

Given a non-empty sequence of integers where exactly one element appears once
and all others may appear multiple times, find the element that appears exactly once.

## Input

- `a`: a non-empty sequence with at least one element having count 1

## Output

An integer `unique` such that it appears in the array and has count exactly 1.

## Examples

- `SingleNumber([2, 2, 1])` returns `1`
- `SingleNumber([4, 1, 2, 1, 2])` returns `4`
- `SingleNumber([42])` returns `42`
