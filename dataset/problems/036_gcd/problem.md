# Compute GCD (Euclidean Algorithm)

## Description

Compute the greatest common divisor of two positive integers using the Euclidean algorithm.

## Input

- `a`: a positive integer
- `b`: a positive integer

## Output

An integer `g` equal to `GcdFunc(a, b)` (the recursive GCD), which is positive.

## Examples

- `Gcd(12, 8)` returns `4`
- `Gcd(7, 7)` returns `7`
- `Gcd(3, 5)` returns `1`
