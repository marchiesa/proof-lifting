# Rain Water Trapping (Two Pointer)

## Description

Given an elevation map (sequence of non-negative integers representing bar heights), compute how much water can be trapped after raining, using the two-pointer technique.

## Input

- `h`: a non-empty sequence of non-negative integers

## Output

A non-negative integer `water` representing the total trapped water.

## Examples

- `TrapWater([0,1,0,2,1,0,1,3,2,1,2,1])` returns `6`
- `TrapWater([4,2,0,3,2,5])` returns `9`
