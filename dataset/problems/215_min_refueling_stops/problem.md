# Minimum Number of Refueling Stops

## Description

A car starts with some fuel and needs to reach a target distance. Along the way there are gas stations at given positions with given fuel amounts. Find the minimum number of refueling stops.

## Input

- `target`: distance to reach
- `startFuel`: initial fuel
- `positions`: sorted sequence of station positions
- `fuels`: fuel available at each station

## Output

Minimum stops, or -1 if impossible.
