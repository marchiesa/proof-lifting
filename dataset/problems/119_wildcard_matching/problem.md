# Wildcard Matching

## Description
Implement wildcard matching with '?' (encoded as 0, matches any single char) and '*' (encoded as -1, matches any sequence of characters).

## Input
- `s`: sequence of positive integers (the string)
- `p`: pattern sequence (positive ints, 0 for '?', -1 for '*')

## Output
`result`: true if the string matches the pattern.
