# Intersection of Two Arrays

## Description

Given two sequences of integers, return a sequence of unique elements
that appear in both arrays.

## Input

- `a`: first sequence of integers
- `b`: second sequence of integers

## Output

A sequence `result` containing unique elements from the intersection.

## Examples

- `Intersection([1,2,2,1], [2,2])` returns `[2]`
- `Intersection([4,9,5], [9,4,9,8,4])` returns a permutation of `[4,9]`
