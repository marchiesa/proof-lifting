# Dot Product

## Description

Compute the dot product of two integer sequences of equal length.

## Input

- `a`, `b`: two sequences of integers with `|a| == |b|`

## Output

The integer `a[0]*b[0] + a[1]*b[1] + ... + a[n-1]*b[n-1]`.

## Examples

- `ComputeDotProduct([1, 2, 3], [4, 5, 6])` returns `32`
- `ComputeDotProduct([1, 0], [0, 1])` returns `0`
- `ComputeDotProduct([], [])` returns `0`
