# Maximum Number of Events That Can Be Attended

## Description

Given events represented as [start, end] pairs, find the maximum number of events that can be attended. You can attend at most one event per day, and you can attend an event on any day in [start, end].

## Input

- `starts`: sequence of event start days
- `ends`: sequence of event end days
- Both have same length, starts[i] <= ends[i]

## Output

Maximum number of events that can be attended.
