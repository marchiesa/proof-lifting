# Prim's Minimum Spanning Tree (Array-Based)

## Description

Given a weighted undirected graph as an adjacency matrix, compute the total weight of a minimum spanning tree using Prim's algorithm.

## Input

- `graph`: adjacency matrix where `graph[i][j]` is the weight of edge (i,j), or -1 if no edge
- `n`: number of vertices

## Output

The total weight of the MST, or -1 if the graph is disconnected.
