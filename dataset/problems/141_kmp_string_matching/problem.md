# KMP String Matching

## Description

Find the first occurrence of a pattern in a text string. Returns the starting
position or -1 if not found. The first match is the earliest one.

## Input

- `text`: sequence of characters
- `pattern`: non-empty pattern to search for

## Output

Position of first occurrence, or -1 if not found.

## Examples

- `KMPSearch("hello", "ll")` returns `2`
- `KMPSearch("hello", "xyz")` returns `-1`
