# Linked List Cycle Detection

## Description

Given a sequence representing a linked list (where each element is the index of the next node, or -1 for end), determine if the list contains a cycle using Floyd's tortoise and hare algorithm.

## Input

- `next`: a sequence of integers where next[i] is the successor of node i, or -1 if no successor
- `start`: the starting node index

## Output

A boolean indicating whether there is a cycle reachable from start.
