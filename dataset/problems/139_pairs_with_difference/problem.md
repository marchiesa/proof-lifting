# Find All Pairs with Given Difference

## Description

Count pairs of indices (i, j) with i < j such that |a[i] - a[j]| equals a given difference.

## Input

- `a`: sequence of integers
- `diff`: non-negative integer difference

## Output

Count of pairs with the given absolute difference.

## Examples

- `FindPairsWithDiff([1, 5, 3, 4, 2], 2)` returns `3`
- `FindPairsWithDiff([1, 2, 3], 1)` returns `2`
