# Majority Element (Boyer-Moore Voting)

## Description

Find the majority element in a sequence using the Boyer-Moore voting algorithm. A majority element appears more than n/2 times. The input is guaranteed to have a majority element.

## Input

- `a`: a non-empty sequence with a majority element

## Output

An integer `candidate` that is an element of `a`.

## Examples

- `FindMajority([3, 3, 3, 2, 1])` returns `3`
- `FindMajority([7, 7, 7])` returns `7`
