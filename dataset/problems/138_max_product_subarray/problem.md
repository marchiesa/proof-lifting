# Maximum Product Subarray

## Description

Find the contiguous subarray within a non-empty sequence that has the largest product.

## Input

- `a`: a non-empty sequence of integers

## Output

The maximum product achievable by any contiguous subarray.

## Examples

- `MaxProductSubarray([2, 3, -2, 4])` returns `6` (subarray [2,3])
- `MaxProductSubarray([-2, 0, -1])` returns `0`
