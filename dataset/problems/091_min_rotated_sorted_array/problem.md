# Find Minimum in Rotated Sorted Array

## Description

Given a non-empty sequence of integers, find the minimum element using a linear scan.
The specification uses a recursive definition of the sequence minimum.

## Input

- `a`: a non-empty sequence of integers

## Output

An integer `min` equal to the minimum element in the sequence.

## Examples

- `FindMin([3, 4, 5, 1, 2])` returns `1`
- `FindMin([1, 2, 3, 4, 5])` returns `1`
- `FindMin([42])` returns `42`
