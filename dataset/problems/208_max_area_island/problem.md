# Maximum Area of Island

## Description

Given a grid (flattened as a 1D array with row/col dimensions) of 0s and 1s, find the maximum area of an island (connected 1s using 4-directional adjacency).

## Input

- `grid`: a sequence of 0s and 1s
- `rows`: number of rows
- `cols`: number of columns

## Output

The maximum island area.
