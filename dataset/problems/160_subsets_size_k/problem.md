# Generate All Subsets of Size K

## Description

Given n and k, generate all subsets of {0, 1, ..., n-1} that have exactly k elements. Return them as a sequence of sorted sequences.

## Input

- `n`: universe size (elements 0..n-1)
- `k`: desired subset size

## Output

A sequence of all k-element subsets, each subset sorted in increasing order.
