# Majority Element

## Description

Given a non-empty sequence of integers, determine if there is a majority element
(an element appearing more than n/2 times). If so, return it.

## Input

- `a`: a non-empty sequence of integers

## Output

- `found`: true if a majority element exists
- `candidate`: the majority element (if found)

## Examples

- `FindMajority([1, 1, 1, 2, 3])` returns `(true, 1)`
- `FindMajority([1, 2, 3])` returns `(false, _)`
