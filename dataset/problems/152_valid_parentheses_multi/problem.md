# Valid Parentheses (Multiple Types)

## Description

Given a string of parentheses characters `(`, `)`, `[`, `]`, `{`, `}`, determine whether the string is valid. A string is valid if every open bracket is closed by the same type of bracket in the correct order.

## Input

- `s`: a sequence of characters (each being one of `()[]{}`)

## Output

A boolean indicating whether the parentheses are balanced and properly nested.
