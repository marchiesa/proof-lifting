# Remove Consecutive Duplicates

## Description

Remove consecutive duplicate elements from a sequence, keeping only the first
occurrence in each run of equal elements.

## Input

- `a`: a sequence of integers

## Output

A sequence with no consecutive duplicates, preserving the first and last elements.

## Examples

- `RemoveConsecutiveDups([1, 1, 2, 2, 3])` returns `[1, 2, 3]`
- `RemoveConsecutiveDups([1, 2, 1])` returns `[1, 2, 1]`
- `RemoveConsecutiveDups([])` returns `[]`
