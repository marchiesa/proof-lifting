# Longest Bitonic Subsequence

## Description

A bitonic subsequence first increases then decreases. Given a sequence, find the length of the longest bitonic subsequence using LIS (Longest Increasing Subsequence) from left and LDS (Longest Decreasing Subsequence) from right.

## Input

- `a`: a sequence of integers with length > 0

## Output

The length of the longest bitonic subsequence.
